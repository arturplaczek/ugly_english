## Getting Started ğŸš€

This project contains 3 flavors:

- development
- staging
- production

To run the desired flavor either use the launch configuration in VSCode/Android Studio or use the following commands:

```sh
# Development
$ flutter run --flavor development --target lib/main_development.dart

# Staging
$ flutter run --flavor staging --target lib/main_staging.dart

# Production
$ flutter run --flavor production --target lib/main_production.dart
```

_\*Ugly English works on iOS, Android and Web._

```sh

$ genhtml coverage/lcov.info -o coverage/

$ open coverage/index.html
```

To work with git crypt, you need to have the key to decrypt the files. After cloning the repository, you can decrypt the files using the following command:

```sh
$ git-crypt unlock /path/to/your/key
```
